## Usage

- Compare files:

`gompare --file <file1_path> <file2_path>`

- Compare file&hash:

`gompare --hash <file_path> <hash>`

## Installation

- Enter [this GitHub repo](https://github.com/lexionq/gompare)

- Download file named `gompare`

- Run this commands:

```bash
chmod +x gompare
sudo mv gompare /usr/bin/gompare
```
